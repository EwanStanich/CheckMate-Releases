# CheckMate — Releases

Public distribution for the **CheckMate** desktop app — a purpose-built tool for manual
test-case management.

This repository holds **built installers only**. The application source is private; this
repo exists solely to host release binaries and update metadata.

## Download

Get the latest version from the download page:

**https://checkmate-app.com/download**

It automatically serves the right installer for your operating system. You can also grab a
specific build directly from the [Releases](../../releases) page.

| Platform | Installer |
|----------|-----------|
| Windows  | `.exe`    |
| macOS    | `.dmg`    |

## Installing

The app is **not code-signed**, so your OS will show a warning the first time you install:

- **Windows** — SmartScreen may say "Windows protected your PC." Click **More info → Run anyway**.
- **macOS** — if you see "unidentified developer," right-click the app → **Open**, then confirm.

## Updates

CheckMate checks this repository for new versions on launch. When an update is available,
the app will prompt you and link back to the download page. You can also check manually via
**Settings → Personal → Check for Updates**.
